# Marketing-Funnel-Analytics-Campaign-ROI-Optimization-MySQL-Power-BI
Diagnosed negative 88% marketing ROI by analyzing 10K users, uncovering 51% funnel leakage and optimizing campaign performance using Power BI &amp; MySQL

This project analyzes the marketing performance of a D2C e-commerce company (NovaGrowth E-Commerce Pvt. Ltd.) to identify funnel leakage, campaign inefficiencies, and ROI gaps.

Despite significant marketing investment, the company faced heavy losses — and this project uncovers why.

**Business Problem**

NovaGrowth spent:

💰 ₹1.12M on marketing
💸 Generated only ₹133.42K revenue
📉 Resulting in –88.07% ROI

**Key Question: **

Where is the leakage in the funnel and how can ROI be improved?

**Key Insights**

**1. Funnel Leakage:**

* 10,000 users entered the funnel
* Only 1,005 converted (10.05%)
* 51.10% drop at Interest → Consideration

Insight: Mid-funnel is the biggest problem, not traffic

**2. Campaign Performance**

* Few campaigns delivered 100%+ ROI
* Majority campaigns had negative ROI

Insight: Budget is not allocated based on performance

**3. Device Analysis**

* Mobile, Desktop, Tablet → all negative ROI

Insight: Issue is not device-specific, but funnel inefficiency

**4. Channel ROAS**

* Mixed performance across Meta, Google, Email, Affiliate

Insight: No structured channel optimization strategy

**Final Diagnosis**

* Not a traffic problem
* Not a device problem
* Conversion + budget allocation problem

**Recommendations**

**1. Budget Reallocation**

Pause low-performing campaigns
Invest in high ROI campaigns
Expected: Reduce wasted spend by 25–35%

**2. Mid-Funnel Optimization**

* Fix 51% drop at Interest → Consideration
* Improve landing page + messaging
* Expected: Conversion ↑ to 14–15%

**3. Retargeting Strategy**

* Target high-intent users (Consideration & Intent)
* Use email + dynamic ads
* Expected: +5–8% conversion lift

**4. KPI Governance**

* Define ROAS benchmarks
* Monthly performance monitoring

**What I Learned**

* Funnel analytics is more powerful than surface-level metrics
* Mid-funnel optimization drives the biggest impact
* ROI > Vanity metrics (clicks, impressions)
* Budget allocation is as important as campaign performance
